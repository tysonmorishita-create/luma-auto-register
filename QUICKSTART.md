### First Use (4 minutes)

#### Step 1: Get a Calendar URL
Go to any Luma calendar page, for example:
```
https://luma.com/consensus?k=c&period=past
```
Copy the URL.

#### Step 2: Scan Events
1. Click the extension icon
2. Paste the calendar URL
3. Click "Scan Events"
4. Wait for events to load

#### Step 3: Select & Configure
1. Check events you want to register for
2. Set parallel tabs: `2` (recommended)
3. Set delay: `5` seconds (recommended)

#### Step 4: Register!
1. Click "Start Registration"
2. Watch the progress bar
3. See results in real-time

#### Step 5: Export Results
1. Click "Export Results" to download CSV
2. Or click "Start New Batch" for more events
